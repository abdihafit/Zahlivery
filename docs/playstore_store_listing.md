## App name

## Short description
Browse shop menus, place orders, chat with shops, and manage rider deliveries.

## Full description
Zahlivery is a multi-role delivery app for customers, shops or hotels, and
riders.

Customers can use Zahlivery to:
- create an account and choose the customer portal
- browse available shops and hotel menus
- view item photos, descriptions, and prices
- place an order from the menu
- chat with the shop about the order
- view order status updates
- contact the assigned rider when rider details are available

Shops and hotels can use Zahlivery to:
- create a shop account
- add menu items with prices, descriptions, and food photos
- receive incoming customer orders
- review order details and totals
- update the order status
- assign an available rider
- chat with customers

Riders can use Zahlivery to:
- create a rider account
- mark themselves available for delivery
- receive assigned orders
- contact customers by chat or phone
- mark deliveries as completed

Zahlivery is best suited for local ordering and delivery coordination where the
customer, shop, and rider all need one shared workflow.
