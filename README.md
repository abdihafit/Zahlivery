# Zahlivery

Zahlivery is a Flutter and Firebase delivery platform that connects customers,
hotels or shops, riders, and admins in one system.

## Features

- Customer ordering and order history
- Hotel or shop menu management
- Rider delivery workflow and chats
- In-app notifications for chats, orders, and deliveries
- Admin dashboard for live platform monitoring

## Stack

- Flutter
- Firebase Authentication
- Cloud Firestore
- Firebase Cloud Functions
- Firebase Storage

## Getting Started

1. Install Flutter and Firebase CLI
2. Run `flutter pub get`
3. Add the correct Firebase config files for your environment:
   - `android/app/google-services.json`
   - `android/key.properties`
   - `lib/firebase_options.dart` if you still use generated options outside Android
4. Run `flutter run`
